Detailed Description

Class Model Container for the neural network (NN).

To avoid runtime errors make sure the labelled classes can be found in the defined classes list (refer to config.py).

Class can be initialized with either an SSD or an FRCNN neural network.

To initialize class with a partially trained neural network, make sure the argument trained is set to True. Remaining configuration variables can be found in config.py.

Parameters
