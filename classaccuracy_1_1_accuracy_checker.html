Detailed Description

Class AccuracyChecker Used to check the accuracy of the model's prediction against the ground truth (labelled benchmark).

Benchmarks must be LabelImg-style xml files. Use the label option when initializing class: Test before initializing this class. Class can be run directly from file, e.g. python AccuracyChecker.py.
