Detailed Description

Class: Test.

Inference class used to evaluate the test data.

Test data should be included in testingPath (refer to config.py).

Class works in a similar manner to callable objects, i.e. initializes the class implicitly calls the private method "__testModel()".

◆ __testModel()

def inference.Test.__testModel (   self)
private

For all JPG files found in testingPath, the function imports the image and evaluates it using the class property model.

If the NN's outputs are non-zero in length (symbols were detected), the function maps the locations back to the original plan's coordinate system (this function assumes that the user ran SyntheticData on the testing dataset) by calling ____mapBackToOriginal(). __updateXML() is then called to add the prediction to the original plan's XML file. If "label" was not set to true, the XML and JPG files of the original plans are imported and the images are overlaid with the predictions of the NN. Here the output of the segmentation model is used as the original plan.

Returns
No return.
